# 全球机器人产业全景数据库 - 项目指南

## 项目概述

React + TypeScript 的机器人产业数据库 Web 应用。使用 Firebase 做认证和后端，Vite 构建。

## 数据架构

### 企业数据三层结构

1. **真实锚点企业**（`generator.ts` 中的 `realAnchors` 数组）
   - 手动录入的全球知名机器人企业
   - 完整字段：名称、英文名、国家、城市、成立年份、产业链环节、类别、产品、融资、估值、评级、营收、产量、技术、投资方

2. **补充企业**（`companies_patch.ts` 中的 `chinesePatchCompanies` 数组）
   - 352 家补充的真实企业，首次追加海外企业时新增了 `co` (country) 字段
   - 使用紧凑字段名：`n/e/ct/co/y/s/c/p/r/v/stars/rev/o/t/i`
   - 通过 `generator.ts` 的合并逻辑与锚点企业去重后统一入库

3. **程序生成企业**（`generator.ts` 中的 `generateEnterprises()` 函数）
   - 15,881 家，使用 `seededRandom()` 确定性算法生成
   - 覆盖中国/美国/日本/韩国/瑞士/英国/法国/新加坡 8 国
   - 生成时按 `nameSet` 去重

### 关键文件说明

| 文件 | 用途 |
|------|------|
| `src/companies_patch.ts` | **主要修改入口**——添加/修改企业数据 |
| `src/generator.ts` | 企业生成引擎，修改生成逻辑或添加锚点 |
| `src/data.ts` | 24 张分类工作表（产业链总图、人形机器人等） |
| `src/App.tsx` | 主应用 UI，企业库、搜索、生态图谱、报告等 |
| `src/data/products.ts` | 产品详细数据（17 个产品 BOM 表） |
| `src/data/reportsAndDynamics.ts` | 研报和行业动态 |

## 添加新企业

编辑 `src/companies_patch.ts`，在数组末尾追加条目：

```typescript
{
  n: "企业名称",
  e: "EnglishName",
  ct: "城市",
  co: "国家", // 可选，默认"中国"
  y: 2024,    // 成立年份
  s: "🟦中游", // 🔴上游/🟦中游/🟢下游
  c: "类别",   // 参考下方类别
  p: "产品",
  r: "融资轮次",
  v: "估值",
  stars: "⭐⭐⭐", // ⭐/⭐⭐/⭐⭐⭐
  rev: "营收",
  o: "2027产量",
  t: "核心技术",
  i: "投资方"
}
```

## 构建与部署

```bash
npm install      # 安装依赖
npm run dev      # 本地开发 localhost:3000
npm run build    # 生产构建，产出 dist/
npm run lint     # TypeScript 类型检查
```

## 注意事项

- `generator.ts` 中的 `seededRandom()` 使用 `Math.sin` 实现确定性伪随机，修改种子值会影响所有生成数据
- 企业名称通过 `nameSet` 去重，重名会自动追加后缀
- 海外企业通过 `co` 字段区分，`generator.ts` 中合并时使用 `x.co || "中国"` 兼容旧数据
- 所有 emoji 标记（🟦🟢🔴⭐）需保持一致
